HistogramAxis¶

class histogram.HistogramAxis(bins, limits=None, label=None)[source]¶

A single axis used internally by Histogram to store the bin edges in an open-grid format.

An axis consists of a continuous range on the real-line, divided into bins such that there are no breaks. The bins do not have to be uniform, but they do have to cover the entire range.

Parameters:
  • bins (int, float array) – Number of bins in combination with
  • or 1D array of edges including the upper limit of the (limits,) –
  • bin. (last) –
  • limits (float 2-tuple) – (min,max) representing the limits of
  • axis used in combination with integer bins. (this) –
  • label (str) – The axis label including units if applicable.
Raises:
  • TypeError – If bins is a list and limits is not None.
  • ValueError – If arguments could not be understood as an axis along the real line.

Examples

The following examples both create an axis of 100 uniform bins from 0 to 10. In this case, there are 101 edges to this axis:

a1 = HistogramAxis(100,[0,10])
a2 = HistogramAxis(np.linspace(0,10,101))
assert a1 == a2

A non-uniform binning example: 10 bins on a log-scale from 1 to 10000:

a3 = HistogramAxis(np.logspace(0,4,11))

HistogramAxis.isuniform(rtol=1e-05, atol=1e-08)[source]¶

Check if all bins are of equal width.

Parameters:
  • rtol (float) – relative tolerance parameter
  • atol (float) – absolute tolerance parameter
Returns:

True if all bins are the same size
Return type:

bool

Note

This returns True if all bin-widths satisfy the following:

abs(widths - mean) <= (atol + rtol * abs(median))

The axis is considered uniform if the maximum deviation of the bin-widths is less than the given tolerance:

\[\max\left(\frac{\left|w_{i} - \left<w\right>\right|}{\left<w\right>}\right) < tol\]

where \(\left<w\right>\) is the median bin width and \(max()\) is evaluated over all bin widths, \(w_{i}\).

Compare edges to within numpy’s default tolerance.

Note

Labels are ignored and each point in the edge arrays must satisfy

\[\left(\frac{\left|e - e^{\prime}\right|} {0.5 (e + e^{\prime})}\right) < tol,\]

where \(e := \mathtt{self.edges[i]}\) and \(e^{\prime} := \mathtt{that.edges[i]}\).
